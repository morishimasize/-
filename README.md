# うちの献立アプリ 🍳

家族4人（大人2人＋小学生の女の子2人）の1週間分の夕飯を自動で考えてくれるPWA。
iPad・iPhoneのホーム画面に追加して使う。

---

## ✨ 機能

- **「今週食べたいもの」入力** — 入れた料理は必ずその週に組み込まれる
- **🎲 1週間献立の自動生成** — 定番＋旬の食材を組み合わせて毎回違うメニュー
- **レシピ表示** — タップで材料（4人分）と作り方を表示
- **🧊 冷蔵庫レシピ生成**（v1.5〜）
  - 専用タブで今ある食材をチップ式で入力
  - AIが**基本調味料のみ**（醤油・みりん・酒・砂糖・塩・味噌・酢・油など）で作れる時短レシピを生成
  - 市販の素・ミールキットは一切使わない「残り物アレンジ」スタイル
  - 生成されたレシピはそのままメニューDBに保存可能
- **✨ AIレシピ自動生成**（v1.3〜）
  - 「食べたいもの」に未登録の料理を入力すると「✨ AIでレシピを生成する」ボタンが出る
  - タップするとClaude（Haiku 4.5）が材料（4人分）と手順を自動生成
  - **もりもり家スタイル**（v1.3.1〜）：市販の素・ミールキット・インスタント食材を優先した時短レシピを生成
  - 生成されたレシピは買い物リストにも即反映、メニューDBにも永続保存
  - APIキーは設定タブで登録（localStorageに保存、サーバー送信なし）
- **✨ メニュー追加時のAI自動入力**（v1.4〜）
  - メニュータブの「＋追加」で料理名を入れ → 「✨ AIでレシピを自動入力」をタップ
  - もりもり家スタイル（市販品活用・時短）で材料・手順が自動入力される
  - 内容を確認・編集してから「保存」できる
- **📤 カスタムメニューの家族共有**（v1.4〜）
  - メニュータブ上部「📤 共有」ボタンで、自分で追加したメニューをAirDropなどで送信
  - デフォルトの23品は省略、custom-IDのメニューのみ共有
  - 受け取り側はURLを開くと「📦 カスタムメニューが届いたよ！」ダイアログが表示
  - 「追加する」を押すとメニューDBにマージ（同名は自動上書き）
- **📤 献立の家族共有**（v1.2〜）
  - 「📤 この献立を家族に共有」ボタンでiOS共有シートからLINE・メッセージ等で送信
  - 受け取った側はURLを開くと「📨 共有された献立が届いたよ！」ダイアログが表示
  - 「読み込む」を押すと献立と買い物リストがそのまま使える
  - サーバー不要・完全オフライン対応
- **🛒 賢い買い物リスト**（v1.1〜）
  - 📅 **何日分を買うか曜日で指定**できる（週末まとめ買い／週中追加に対応）
  - 🧊 **冷凍・レトルトで済ます献立**を選んでスキップ
  - 🥬 **家にある食材**をチェックして除外（在庫チェック）
  - 🌻 季節に応じた**買い物のヒント**（夏は3〜4日推奨／冬は1週間OKなど）
  - 同じ食材は数量を自動合算
- **📤 iOSの共有シートで送信** — メモ・リマインダー・LINE・Keepなど好きなアプリへ
- **メニュー編集** — 家庭の味を追加・編集・削除（写真URLも登録可）
- **季節判定** — 自動で春夏秋冬の旬メニュー（鍋・冷やし中華など）を組み込む

## 🚀 GitHub Pages への公開手順

### 1. GitHubでリポジトリを作る

1. [github.com](https://github.com) にログイン
2. 右上の「+」→「New repository」
3. リポジトリ名を入れる（例: `kondate-app`）
4. **Public** を選択（Pages は Public 必須／Pro 課金で Private も可）
5. 「Create repository」

### 2. ファイルをアップロード

このフォルダの中身を全部、リポジトリにアップロードする：

```
kondate-app/
├── index.html
├── manifest.json
├── icon-152.png
├── icon-167.png
├── icon-180.png
├── icon-192.png
└── icon-512.png
```

リポジトリのページで「Add file」→「Upload files」→ ドラッグ＆ドロップで全部アップ → 「Commit changes」。

### 3. GitHub Pages を有効化

1. リポジトリの「Settings」を開く
2. 左メニューの「Pages」をクリック
3. 「Build and deployment」セクションで：
   - Source: **Deploy from a branch**
   - Branch: **main** / **(root)** を選んで「Save」
4. 数分待つと、Pages のページ上部に公開URLが表示される
   （例: `https://もりもり.github.io/kondate-app/`）

### 4. iPad/iPhone のホーム画面に追加

1. **Safari** で公開URLを開く（Chromeでは×）
2. 画面下の「共有」ボタン（□↑）をタップ
3. メニューを下にスクロールして「**ホーム画面に追加**」をタップ
4. 名前を確認して「追加」

→ ホーム画面にアイコン🍳が現れて、タップするとフルスクリーンでアプリが起動！

## 📱 使い方

### 献立タブ
1. **今週食べたいもの**があれば「追加」ボタンで入れる（例：餃子、ラーメン）
2. 真ん中の大きい **「🎲 1週間分の献立を考えて！」** を押す
3. 月〜日の7日分が表示される
   - ⭐ オレンジ枠 = 今週の希望
   - 🌿 緑枠 = 旬のもの
   - 黄色枠 = 通常メニュー
4. 各カードをタップ → 材料と作り方を表示
5. **未登録の希望料理**には「✨ AIでレシピを生成する」ボタンが出る（APIキー設定済みの場合）
6. **「📤 この献立を家族に共有」** でURLを生成 → 共有シートからLINE等で送信
7. 下の **「🛒 買い物リストを見る」** で買い物リスト画面へ

### 🧊 冷蔵庫レシピタブ（v1.5〜）
1. 冷蔵庫にある食材を入力欄に打ち込んで「追加」（Enterキーでも追加可）
2. 食材がチップとして追加され、入力欄は自動でクリアされカーソルが戻る
3. 3品以上入れると精度が上がる
4. **「🍳 レシピを考えて！」** をタップ → AIが数秒でレシピを生成
5. 材料・手順を確認して、気に入ったら **「📚 メニューに保存する」** でDB登録

> **冷蔵庫レシピのポイント：** 醤油・みりん・酒・砂糖・塩・味噌・酢・サラダ油・ごま油・にんにく・生姜など**家にある基本調味料だけ**で作れるレシピを生成。市販の素やミールキットは使わないので、買い物なしで作れます。

### AIレシピ自動生成の使い方
1. 「設定」タブを開く
2. 🔑 **APIキー入力欄**にAnthropicのAPIキーを貼り付けて「💾 保存する」
   - APIキーは [console.anthropic.com](https://console.anthropic.com) で取得
   - ⚠️ コンソールの「自動リロード」は**オフ**にしておくと安心（$5上限で止まる）
3. 「献立」タブに戻り、食べたいものに未登録の料理を入力して献立を生成
4. 「✨ AIでレシピを生成する」ボタンをタップ → 数秒で材料・手順が完成！
5. 生成されたレシピは買い物リストにも自動反映、次回以降もメニューとして使える

> **もりもり家スタイル（v1.3.1〜）：** 市販の素・ミールキット・インスタント食材を積極活用した時短レシピで生成される。例：ラーメン系は3食入り生麺スープ付きセット、麻婆豆腐は市販の麻婆豆腐の素を使ったレシピ。

> **APIキーについて：** claude.aiのアカウント（Pro/無料）とは別物。誰のデバイスで開いても同じキーを使う。APIキーはそのデバイスのlocalStorageにのみ保存され、外部には送信されない。

### メニュー追加時のAI自動入力（v1.4〜）
1. 「メニュー」タブを開く
2. 「＋ 追加」をタップ
3. 料理名と絵文字を入力
4. **「✨ AIでレシピを自動入力」** をタップ → 材料・手順が自動入力される
5. 内容を確認・編集してから「保存」

### カスタムメニューを家族に共有する（v1.4〜）
1. 「メニュー」タブの上部「📤 共有」をタップ
2. iOS共有シートから **AirDrop** で家族のデバイスへ送信
3. 受け取り側はSafariでURLを開く
4. 「📦 カスタムメニューが届いたよ！」ダイアログでプレビューを確認
5. 「✅ 追加する」をタップ → 自分のメニューにマージされる（同名は上書き）

> **注意：** デフォルトの23品は全デバイスに最初から入っているため共有対象外。「＋追加」で登録したカスタムメニューのみ共有される。

### 献立を受け取るとき（夫・子どものデバイス）
1. LINEなどで届いたURLを **Safari** でタップして開く（Chromeでは×）
2. 「📨 共有された献立が届いたよ！」ダイアログが表示される
3. 月〜日の献立プレビューを確認
4. 「✅ この献立を読み込む」をタップ
5. 自分のアプリに献立が入り、買い物リストもそのまま使える 🎉

> **注意：** 受け取り後は各デバイスで独立して動く。「買った」チェックは同期されない。

### 買い物リスト（3ステップで作る）

#### ① 何日分を買う？
画面上部の月〜日の曜日チップから、買いに行く日を**タップで選択**。

#### ② 冷凍・レトルトで済ますものは？
選んだ日の献立がリストで出るので、**買わずに済ませる料理にチェック**。

#### ③ 家にあるものは？（任意）
「▼ 在庫チェックを開く」をタップして、**すでに家にある食材にチェック**。

#### ④ 買い物リスト
①②③で絞り込まれた食材だけが**自動集計**される。

### メニュータブ
- 登録されてる全メニュー一覧
- タップで編集（料理名・絵文字・写真URL・季節・材料・手順）
- 「+ 追加」で新しい家庭の味を追加（AI自動入力あり）
- 「📤 共有」でカスタムメニューをAirDropなどで家族に送れる

### 設定タブ
- 🔑 **APIキー設定** — AIレシピ自動生成・冷蔵庫レシピ用のAnthropicキーを登録
- 季節を手動上書き（夏に「冬」を強制したい時など）
- データのリセット（追加メニュー・希望・献立が全部消える）

## 💾 データの保存場所

すべてブラウザの localStorage に保存。サーバーには一切送らない（プライベートで安心）。
ただしブラウザ・端末ごとに別データなので、iPad と iPhone で自動同期はしない。
献立の共有は「📤 この献立を家族に共有」ボタンからURLを送る方式。
カスタムメニューの共有は「メニュー」タブの「📤 共有」ボタンからAirDrop等で送る方式。
冷蔵庫レシピで生成したメニューも「メニューに保存」でlocalStorageに保存される。

## 🛠 カスタマイズアイデア

- 写真を追加したい → メニュー編集で「写真URL」に画像のURLを入れる
  （Google フォトの共有リンク・Imgur・自分のサイト等）
- 「お米」を毎回買い物リストに入れたい → メニュー編集で各メニューに追加
- 苦手な食材があるメニューは削除しちゃってOK
- 在庫チェックを毎週使うなら、よく余る食材（玉ねぎ・卵・お米など）から確認すると早い
- AIで追加したメニューを家族に配りたい → メニュータブ「📤 共有」→ AirDrop
- 冷蔵庫レシピで気に入ったものは「メニューに保存」しておくと献立生成にも使われる
